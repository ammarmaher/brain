# Source of Truth Priority

1. Architecture Wiki / architect-approved decisions
2. Backend controllers, DTOs, validators, gateway for API truth
3. Approved PRDs and business decisions
4. Existing codebase and current branch
5. Falcon registries and approved pattern memory
6. HTML / React / screenshot sources
7. Best-practice assumptions

For conversion tasks:

| Source | Role |
|---|---|
| HTML | Visual source of truth |
| React | Behavior/component-structure reference |
| Screenshot | Visual validation reference |
| Backend | API/DTO/validation source of truth |
| Wiki | Architecture source of truth |

If sources conflict, report the conflict and request decision only if blocking.
