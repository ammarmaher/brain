| Intent phrase | Route |
|---|---|
| Initialize / setup brain / first scan | Initial Bootstrap Discovery |
| UI/UX is finished, link backend | Backend API Understanding + Integration |
| Convert HTML | HTML to Falcon Angular |
| Convert React | React to Falcon Angular |
| From screenshot | Screenshot to Falcon Angular + Visual Parity Loop |
| Analyze PRD / PDR | Business Understanding |
| Add validation | Validation Rules |
| PES / permission | PES Permission Analysis |
| Make report / PDF | Task Report Generator |
| Component missing ability | Component Capability Upgrade |
