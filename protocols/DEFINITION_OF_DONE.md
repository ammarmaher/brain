# Definition of Done

A task is complete only when:

1. Role and mode are detected.
2. Branch/repo context is clear.
3. Source of truth is identified.
4. Incremental scan is completed or confirmed unchanged.
5. Architecture wiki is checked.
6. Business/API/component impacts are documented.
7. Falcon components are reused or upgraded when needed.
8. Frontend structure governance is followed.
9. Build/lint/test/visual checks pass or remaining gaps are documented.
10. Visual parity loop is completed for UI tasks.
11. Report is generated.
12. Brain artifacts are committed and pushed.
13. Implementation code is committed and pushed after quality gates when applicable.
