## Status values

- `Active` — module is live, dossier maintained
- `Planned` — designed but not yet implemented
- `Deprecated` — being phased out
- `Archived` — removed from product

## How to add a module

1. Create `modules/<slug>/` with all 7 required files:
   - `README.md`, `behavior.md`, `surface-map.md`, `decisions.md`, `open-questions.md`, `links.md`, `coverage.md`
2. Use [resources/dossier-template.md](./resources/dossier-template.md) for `README.md`
3. Add a row to this INDEX
4. Validate every term against `domain-glossary`
5. Confirm every link in `links.md` resolves

## Regeneration

This INDEX is regenerated on any module add, rename, or status change. Sort alphabetically by slug.
