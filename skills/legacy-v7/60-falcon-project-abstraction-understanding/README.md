# Falcon Project Abstraction Understanding Skill

This Claude Skill helps Claude understand the Falcon front-end project from an architecture and abstraction level before making implementation changes.

## Suggested install location

Place this folder inside your Claude skills/brain folder, for example:

```text
.claude/skills/falcon-project-abstraction-understanding/
```

## Main file

```text
SKILL.md
```

## Recommended use

Tell Claude:

```text
Before implementing anything in the Falcon project, load and follow the falcon-project-abstraction-understanding skill. First explain the abstraction level, ownership, affected apps/libs, existing patterns, risks, and safe implementation plan. Do not start coding until this understanding is clear.
```
