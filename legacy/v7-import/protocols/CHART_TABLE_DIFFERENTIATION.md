# Chart and Table Differentiation Protocol

## Rule

ChatGPT/Codex and Gemini both work with data, but they do not do the same job.

## ChatGPT/Codex owns tables and business matrices

Use ChatGPT/Codex for:

- Business test tables
- Permission matrices
- Status transition matrices
- API mapping tables
- DTO mapping tables
- Acceptance criteria tables
- Regression risk tables
- Decision tables
- Data-to-chart specification tables

ChatGPT/Codex creates the **logical specification**.

## Gemini owns visual charts and chart validation

Use Gemini for:

- Chart screenshots
- Dashboard images
- Visual chart analysis
- Chart type validation
- Axis/legend/tooltip validation
- Visual spacing and layout
- Responsive chart behavior
- Before/after UI comparison

Gemini creates the **visual interpretation and validation**.

## Claude owns chart implementation

Claude implements charts only after receiving:

- Chart type
- Data source
- Data shape
- Axis fields
- Series/grouping
- Tooltip rules
- Empty/loading/error states
- Theme rules
- Acceptance criteria
