# Gemini Visual QA Checklist

Gemini should verify:

- [ ] Layout matches requirement.
- [ ] Spacing is consistent.
- [ ] Alignment is correct.
- [ ] No unnecessary horizontal scroll.
- [ ] Text truncation has tooltip if needed.
- [ ] Charts have correct type.
- [ ] Axes are correct.
- [ ] Legend is correct.
- [ ] Tooltip is correct.
- [ ] Filters update chart correctly.
- [ ] Loading state exists.
- [ ] Empty state exists.
- [ ] Error state exists.
- [ ] Responsive behavior is acceptable.
- [ ] Accessibility issues are documented.
- [ ] Implementation matches original screenshot/chart intent.
