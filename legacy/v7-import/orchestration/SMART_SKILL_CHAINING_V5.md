# Smart Skill Chaining v5

## Principle

Do not load every skill blindly. Load the smallest correct chain.

## Chains

### React to Angular

```txt
Architecture → React-to-Angular RAGE MODE → Component Mapping → PES if needed → Validation if needed → Retest/Restructure/Redraw
```

### HTML/Screenshot to Angular

```txt
Architecture → HTML-to-Falcon Angular RAGE MODE → Component Mapping → PES if needed → Retest/Restructure/Redraw
```

### PRD to Sprint

```txt
PRD Intelligence → Business Rules → PES Matrix → Validation Matrix → Frontend/Backend Traceability → Sprint Plan → Artifact Governance
```

### Validation Feature

```txt
Business Validation → PES if visibility/access involved → Frontend behavior → Backend/API behavior if needed → Tests → Retest/Restructure/Redraw
```

### Backend Work

```txt
Backend Architecture → Business Rule → API/DTO/Service → Frontend Traceability if needed → Test Matrix
```

## Question Rule

Ask only when the missing information affects correctness.

Use:

```txt
Question:
Why it matters:
Default safe assumption:
Affected areas:
```
