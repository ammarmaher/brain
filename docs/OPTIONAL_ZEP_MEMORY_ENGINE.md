# Optional Zep / Graph Memory Engine

Brain SK does not require Zep/Graphiti/Neo4j for the August delivery target.
Use Git Markdown + Obsidian + scan metadata first.

Add a graph memory engine later only if:

- Markdown registries become too large.
- You need relationship queries across many modules.
- You need temporal history of business rule changes.
- You need impact analysis like “what breaks if this rule changes?”

Until then, keep graph memory as optional future architecture.
