# Brain SK v0.1 Config

Copy `brain.config.example.json` to `brain.config.json` in this folder and adjust only if your local paths change.

Default local layout:

```text
C:\Falcon\Brain SK        = Brain SK core package / Git repo
C:\Falcon\Falcon          = active Falcon project root with frontend/backend
C:\Falcon\Brain Outputs   = generated understanding, reports, scan metadata
C:\Falcon\Brain SK\_obsidian = Obsidian vault/index
```

Generated understanding files should stay outside the core brain while working, then mirror to `C:\Falcon\Brain SK\outputs` before Git push.
