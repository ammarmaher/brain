# Wallet Type

## English
- **Definition:** Half of the per-Account wallet-topology configuration. Enum: `Single | Multiple`. Determines whether a balance-holder (User or Node, per [[Balance Type]]) has one wallet or multiple wallets (typically one per CommChannel). Configurable by Falcon usertype only.

## Common confusions
- **Wallet Type ↔ Balance Type** — Orthogonal axes of the 2×2 matrix. Wallet Type = how many wallets per holder (Single/Multiple); Balance Type = who holds them (User/Node).
