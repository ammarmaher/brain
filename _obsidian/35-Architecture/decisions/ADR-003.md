# ADR-003 — Why Module Federation (3 apps)

> **Status:** accepted · **Reversal cost:** very-high · **Decided:** 2026-05-16

## TL;DR

One Nx workspace, three federated Angular apps:
- `host-shell` (port 4200, host + auth + layout)
- `admin-console` (port 4204, remote)
- `management-console` (port 4301, remote)

Webpack Module Federation manifest-driven runtime loading. Identical share-map across all 3 apps. `@nx/module-federation@22.7.1` + `@module-federation/enhanced@^2.1.0`.

**Trade-offs accepted:** refresh-token race (D-2026-05-16-14), residual zone.js in package.json, stale duplicate services (D-2026-05-16-17), MockAuth fallback for standalone-remote dev.
