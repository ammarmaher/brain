# ADR-007 — Why @theme over @config

> **Status:** accepted · **Reversal cost:** low · **Decided:** 2026-05-16

## TL;DR

Tokens are declared in CSS-native `@theme` blocks inside `libs/falcon-theme/src/falcon-tailwind-tokens.css`. The `@config` directive is NOT used as a token source. (One surviving `@config` reference at line 8 is a v3→v4 compatibility bridge forwarding `important: true`, not a token source.)

**Verification:** 218 tokens in @theme · 47 per-component `*.tokens.css` files · 0 `@config` for token declaration anywhere in libs/.

**Open question (G-06):** Should `--falcon-leading-*` be renamed to `--leading-*` so Tailwind v4 auto-promotes it?
