# Account Limitations

## English
- **Definition:** Numeric limits on an Account: `maxNormalUserLimit`, `maxSystemUserLimit`, `maxNodeLevels`, `balanceTransferLimitPct`. Special rule: `0` = no limit; empty values are not allowed. Counters for Normal Users and System Users are independent.
