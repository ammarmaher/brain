# Visibility

## English
- **Definition:** A two-value toggle (`Hide | Show`) on every `CommChannelConfig` and `AppConfig` that controls whether the channel/app is exposed to the client side. Default = `Hide`. When set to `Show`, Pricing Type and Price Value become mandatory.
