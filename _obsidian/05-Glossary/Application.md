# Application (App)

## English
- **Definition:** A higher-level Falcon product (e.g. AI, Nabaa) offered to an Account. Each Account has a per-app `AppConfig` with the same shape as `CommChannelConfig`: Visibility, Pricing Type, Price Value, lifecycle status.
