# Node Admin (NA)

## English
- **Definition:** A client-side admin user scoped to a Sub-Node only. Can manage users + sub-nodes inside their scope and perform wallet transfers within their scope. Cannot edit Account-level settings, CommChannel pricing/visibility, or App-level config.
