# Falcon Checkbox Group

> Angular layout + multi-binding wrapper around `<falcon-angular-checkbox>` instances. Manages a `selectedValues` array. Active / preferred for inline / stacked option lists ≤ ~12 entries. Long lists should use `<falcon-angular-multi-select>` instead.
