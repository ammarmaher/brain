# Falcon Calendar

> Single-month inline calendar grid (Stencil + Angular wrapper) with date selection, min/max bounds, disabled-dates predicate, customizable first-day-of-week / locale / week numbers. Used standalone for inline date selection or composed inside `<falcon-angular-date-picker>` for the popover variant. Distinct from the legacy `<falcon-calendar>` facade — see [[Falcon Calendar Legacy]].
