# Falcon Tag

> Dismissible chip / tag with optional leading icon and optional `✕` button. 7-value severity vocabulary (success / info / warning / warn legacy / danger / secondary / contrast). Sibling of `<falcon-badge>` and [[Falcon Status Badge]]. Wave 9.F. Replaces PrimeNG `<p-tag>`.
