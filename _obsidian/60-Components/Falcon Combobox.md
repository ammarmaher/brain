# Falcon Combobox

> Free-text-plus-suggestions combo box. Users can type free text (if `allowFreeText`) or pick from filtered suggestions. Distinct from `<falcon-angular-dropdown>` (closed list, no free text) and `<falcon-angular-multi-select>` (multi-value). Wave 9.H. Needs form-control upgrades (helperText / errorMessage / state / disabled) before broad rollout.
