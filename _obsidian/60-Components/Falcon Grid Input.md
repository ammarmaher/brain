# Falcon Grid Input

> In-grid cell editor with grid-friendly keyboard semantics — Enter commits, Escape cancels (reverts to `originalValue`), Tab/Shift+Tab commits + navigates. Auto-focus on mount. Specialized composed input per architect §5.12.2 (Wave 5). Used by `<falcon-angular-data-table>` and `<falcon-angular-table>` for inline cell editing.
